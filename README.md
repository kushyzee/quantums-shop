# Quantum's Shop

Modern Next.js storefront built with the App Router, TypeScript, and Tailwind CSS v4. Includes animated UI, iconography, and a carousel.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn styles
- Lucide icons
- Motion

## Prerequisites

- Node.js 18.18+ (20+ recommended)
- npm (or pnpm/yarn/bun)

## Getting Started

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

### Build

```bash
npm run build
```

### Start (production)

```bash
npm start
```
